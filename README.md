# 🌐 Network Troubleshooting Lab

## 📋 Overview

This project documents hands-on network troubleshooting exercises performed in a Windows environment. The lab focuses on diagnosing and resolving common connectivity issues using built-in networking tools and troubleshooting methodologies commonly used by IT Support, Help Desk, and Desktop Support professionals.

Through this project, I developed practical experience identifying network-related issues, verifying system configurations, and restoring connectivity using standard Windows networking utilities.

---

## 🛠️ Technologies Used

- Windows 11
- Windows Server 2022
- TCP/IP
- DNS
- DHCP
- Command Prompt
- Oracle VirtualBox

---

## 🎯 Objectives

- Verify IP configuration
- Troubleshoot DNS resolution
- Test network connectivity
- Identify gateway issues
- Validate DHCP configuration
- Diagnose internet connectivity problems
- Utilize Windows networking tools

---

## 🔧 Tasks Completed

### 🌍 IP Configuration

Verified IPv4 configuration using Windows networking tools to ensure proper IP addressing and gateway configuration.

### 📡 Connectivity Testing

### 🌐 DNS Troubleshooting

### 🛠️ Network Diagnostics

### ✅ Connectivity Verification
